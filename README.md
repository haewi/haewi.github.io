## 인공지능이란?
__AI__ : Artificial Intelligence (Machine Intelligence)   
+ 인간의 지능을 인공적으로 만든 것
+ 주로 컴퓨터에 적용한다고 가정

### 인공지능
1. 그저 패턴이 많은 인공지능
2. 데이터를 바탕으로 기계학습하는 인공지능
3. 기계학습을 할 때의 입력값의 특징을 기계가 직접 학습

### 기계학습
→ 코드로 명시하지 않은 동작을 데이터로 학습하여 실행할 수 있는 알고리즘을 개발하는 연구 분야
